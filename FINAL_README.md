📊 Sales & Performance Analytics Dashboard (Power BI)
💡 Project Overview
An end‑to‑end Power BI development project covering:

🧩 Data Modeling (Star Schema)

🧮 DAX Measures & Calculations

📅 Time Intelligence Analysis

📊 Interactive Dashboard Design

🎛️ Advanced Filtering & Navigation

Goal: Transform raw data into actionable business insights that drive smarter decisions.

🗂️ Dataset Structure
Tables used in the model:

Date_Dim

Customer_Dim

Product_Dim

Region_Dim

Sales_Fact

Returns_Fact

🧩 Data Modeling
Designed Fact–Dimension relationships

Applied Primary (PK) and Foreign Keys (FK)

Followed Star Schema principles for simplicity and performance

✅ Ensured:

Clean, consistent structure

Logical naming conventions

Optimized relationships for query efficiency

🧮 DAX Measures
Built dynamic measures using:

CALCULATE, FILTER, ALL, SUMX, COUNTX, AVERAGEX

🧠 Calculated Columns
Profit Margin Category → High / Medium / Low

Customer Full Name → FirstName & " " & LastName

Year‑Month Label → FORMAT(Date, "YYYY‑MMM")

📅 Time Intelligence
Implemented advanced date‑based analytics:

📈 YOY Growth (Year‑over‑Year)

📊 MOM Growth (Month‑over‑Month)

📆 YTD Sales & Returns (Year‑to‑Date)

🔍 Identified seasonal patterns and performance cycles.
